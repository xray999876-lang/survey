แบบสำรวจความพึงพอใจของผู้รับบริการและผู้มีส่วนได้ส่วนเสียต่อคุณภาพการให้บริการ

ความเห็นของท่านช่วยให้เราปรับปรุงบริการได้ตรงจุด ใช้เวลาไม่ถึง 2 นาที

ส่วนที่ 1

ข้อมูลทั่วไปของผู้ตอบแบบสอบถาม

ความพึงพอใจต่อกระบวนการ / ขั้นตอนการให้บริการ

1. มีขั้นตอนการปฏิบัติงานที่ชัดเจน ถูกต้องตามระเบียบ กฎหมาย

2. ให้บริการอย่างเป็นธรรม ตามลำดับก่อนหลังของผู้มารับบริการ

3. ระยะเวลาในการให้บริการมีความเหมาะสม รวดเร็ว

4. มีกระบวนการวิธีการในการตรวจสอบความถูกต้อง ครบถ้วนของข้อมูล

ด้านเจ้าหน้าที่ / บุคลากรที่ให้บริการ

1. มีความรู้ ความสามารถในการให้บริการ เช่น การตอบคำถาม ชี้แจงข้อสงสัย ให้คำแนะนำ ช่วยแก้ไขปัญหาได้อย่างถูกต้อง น่าเชื่อถือ

2. ความเต็มใจและความพร้อมในการให้บริการ

3. ความสุภาพและกิริยามารยาทของเจ้าหน้าที่ผู้ให้บริการ

4. ความซื่อสัตย์สุจริตในการปฏิบัติหน้าที่ เช่น ไม่ขอรับสิ่งตอบแทน ไม่รับสินบน

5. ความเหมาะสมของการแต่งกาย บุคลิกลักษณะท่าทางของผู้ให้บริการ

ด้านสิ่งอำนวยความสะดวกและช่องทางการติดต่อสื่อสาร

1. มีป้าย / สัญลักษณ์ / ประชาสัมพันธ์ บอกจุดบริการ

2. สถานที่ให้บริการสะดวกสบาย มีที่พักผ่อนหย่อนใจ

3. มีช่องทางสื่อสารข้อมูลข่าวสาร ที่หลากหลาย

4. มีช่องทางรับฟังความคิดเห็นต่อการให้บริการ เช่น กล่องรับความคิดเห็น แบบสอบถาม

5. ความสะอาดของสถานที่ให้บริการในภาพรวม

ส่วนที่ 3

ข้อคิดเห็น / ข้อเสนอแนะ

คำตอบของท่านไม่ระบุตัวตน และใช้เพื่อพัฒนาบริการเท่านั้น
